== Description ==
Contactum is drag and drop contact form builder plugin which will allow showing contact form anywhere on the website.

== Features ==
* Drag and Drop Form Builder
* AJAX Submission and Both Client-Side and Server-Side Data Validation for contact forms
* Entry restrictions 
* Expiry time
* Form submission notification
* Export/import forms
* Export form entries
* Elementor Form Widget
* Wordpress Widget
